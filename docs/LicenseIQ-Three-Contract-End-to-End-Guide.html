Rule 4 - Royalty Rate (Standard 5.5%)

{
  "type": "percentage",
  "baseRate": 0.055,
  "basisField": "netSalesValue",
  "tiers": [
    { "min": 0, "max": 99999999.99, "rate": 0.055 }
  ]
}

Rule 5 - High-Volume Royalty Reduction (4.5%)

{
  "type": "percentage",
  "baseRate": 0.045,
  "basisField": "netSalesValue",
  "tiers": [
    { "min": 10000000, "max": 99999999.99, "rate": 0.045 }
  ]
}
Reduced rate triggers when annual net sales reach $10,000,000+

Rule 6 - Minimum Annual Royalty ($500K Floor)

{
  "type": "minimum_guarantee",
  "baseRate": 500000,
  "minimumGuarantee": 500000,
  "description": "Minimum Annual Royalty of $500,000"
}
If calculated royalty is less than $500,000, the fee becomes $500,000.
